# Arbora

A syncretic, non-theistic religion of interconnection, deep time, and presence.

This is a living project. The canon is never closed.

## Structure

```
arbora/
├── index.html          # Homepage
├── cosmology.html      # Cosmology & worldview
├── principles.html     # The Eight Principles
├── practices.html      # Daily practices
├── rituals.html        # Seasonal & community rituals
├── canon.html          # Curated texts & sources
├── tree.html           # The full tree structure
├── src/
│   ├── css/style.css   # Stylesheet
│   └── js/main.js      # Minimal JS
├── docs/
│   └── CHARTER.md      # Hidden charter/dispatch document (NOT published)
└── README.md
```

## For Claude Code Agents

**Before working on any part of this site, read `docs/CHARTER.md` first.** It contains the full context of the founding conversation, the tree structure with detailed node descriptions, instructions for expanding each branch, the tiered curation model, and design principles.

Key rules:
- **Curate, don't invent.** Pull from existing traditions.
- **The canon is never closed.** But it must be lean. Use the three-tier system.
- **Tree structure is sacred.** All content fits within the tree.
- **Multiple genres, not one voice.**
- **Loose practices, not rigid prescriptions.**
- **Use the abstract definition (Part 2 of CHARTER.md) as a diagnostic tool.**

## Deployment

This is a static site designed for GitHub Pages. No build step required—just push to a repo and enable GitHub Pages from the root directory.

## Iteration Protocol

1. Experiment with practices
2. Provide feedback on what worked and what didn't
3. Update relevant branches
4. The tree grows
